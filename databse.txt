2)Find all the topics and tasks which are taught in the month of October:



vbnet
Copy code
SELECT t.name AS topic_name, ts.description AS task_description
FROM topics t
JOIN tasks ts ON t.id = ts.topic_id
JOIN codekata ck ON ts.codekata_id = ck.id
WHERE MONTH(ck.start_date) = 10



3)Find all the company drives which appeared between 15 oct-2020 and 31-oct-2020:



sql
Copy code
SELECT *
FROM company_drives
WHERE start_date >= '2020-10-15' AND end_date <= '2020-10-31'



4)Find all the company drives and students who appeared for the placement:




vbnet
Copy code
SELECT cd.name AS company_drive_name, u.name AS student_name
FROM company_drives cd
JOIN attendance att ON cd.id = att.codekata_id
JOIN users u ON att.user_id = u.id



5)Find the number of problems solved by the user in codekata:




sql
Copy code
SELECT COUNT(*)
FROM attendance att
JOIN tasks ts ON att.codekata_id = ts.codekata_id
WHERE att.user_id = [user_id] AND att.is_present = 1




6)
Find all the mentors with who has the mentee's count more than 15:
vbnet
Copy code
SELECT m.name AS mentor_name, COUNT(mt.id) AS mentee_count
FROM mentors m
JOIN mentees mt ON m.id = mt.mentor_id
GROUP BY m.id
HAVING COUNT(mt.id) > 15
